# SAR Image Reconstruction

## 📌 Project Overview
This project implements Synthetic Aperture Radar (SAR) image reconstruction using Python in Google Colab. The system processes SAR data and reconstructs high-resolution images using signal processing techniques.

## 🚀 Features
- SAR signal preprocessing
- Image reconstruction algorithm
- Data visualization
- Implemented using Google Colab

## 🛠 Technologies Used
- Python
- NumPy
- Matplotlib
- SciPy
- OpenCV

## 📂 Project Files
- SAR_image_reconstruction.ipynb – Main project notebook

## ▶ How to Run (Google Colab)

1. Open the notebook in Google Colab.
2. Run all cells sequentially.
3. Make sure required libraries are installed.

## ▶ How to Run (Local System)

1. Clone the repository:
   git clone https://github.com/Prani-05/SAR-image-reconstruction.git

2. Go to the project folder:
   cd SAR-image-reconstruction

3. Install dependencies:
   pip install -r requirements.txt

4. Open the notebook:
   jupyter notebook
